# Security Policy

If you discover a security vulnerability, please report it privately to the
maintainer (see GitHub profile contact). Do NOT open a public issue for
security-sensitive information.

Preferred reporting steps:
1. Email the maintainer with a subject line `SECURITY: <short desc>`.
2. Include steps to reproduce, impact, and suggested mitigation if known.
3. Allow maintainers reasonable time to respond and fix before public disclosure.

We will respond within 7 business days and coordinate disclosure.
