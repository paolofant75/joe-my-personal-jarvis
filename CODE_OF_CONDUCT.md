# Code of Conduct

This project follows a standard Contributor Covenant-style Code of Conduct.
All participants are expected to be respectful, constructive, and inclusive.

Unacceptable behavior includes harassment, hateful speech, threats, and
disruptive conduct. Maintain professional communication and be mindful of
diversity and inclusion.

Report violations to the repository maintainers via GitHub or email; repeat
offenders may be banned from the project.
